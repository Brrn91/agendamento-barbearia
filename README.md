## 📝 Sobre o Projeto

Este projeto consiste em um sistema de agendamento para barbearia desenvolvido com **HTML, CSS e JavaScript puro**, simulando uma aplicação real utilizada para organização de horários e serviços.

Mais do que um simples projeto, ele foi construído como um laboratório prático para aplicar conceitos de desenvolvimento front-end e lógica de negócio.

---

## 🎯 Objetivo

Aplicar na prática:

- Manipulação do DOM com JavaScript
- Organização de interfaces com HTML e CSS
- Lógica de agendamento (datas, horários e disponibilidade)
- Separação de responsabilidades no código

---

## 🧠 Funcionalidades

### 👤 Área do Cliente

- Página de **login**
- Escolha de **serviços**
- Seleção de **data e horário**
- Confirmação de agendamento - Em desenvolvimento

### 🛠 Painel Administrativo

- Controle geral dos agendamentos - Em desenvolvimento

## 🛠️ Stack Tecnológica

### **Front-end**

- HTML5
- CSS3
- JavaScript (ES6+)

### **Arquitetura**

- Aplicação baseada em manipulação de DOM
- Dados simulados localmente (sem backend)

## ⚙️ Como Executar

Clone o repositório:

```bash

```

Acesse a pasta:

```bash

```

Execute o projeto:

- Abra o arquivo `index.html` no navegador
